Installationsanleitung:
Ein nachträglicher Export für das Eclipse IDE ware leider nicht möglich.
Darum habe ich die verschieden Netbeans-Install auch auf diese CD-ROM
gebrannt.
Netbeans 7.0 für die entsprechende Plattform installieren.
Netbeans starten auf "File" > "Open Project" gehen und dann den Ordner
"java_application" öffnen. 

Anschließend auf "Run" > "Run Main Project" drücken, 
die Java Anwendung start.

--

Eine Quellcodedokumentation ist zufinden unter
"java_application/dist/javadoc/index.html" 

----
Die schriftlich Arbeit ist unter "seminararbeit_latex" zufinden.
